Типы селекторов в CSS:

  • По названию элемента
  • По ID элемента
  • По классу элемента
  • С помощью нескольких классов
  • По вложенности
  • Псевдоклассы

По названию элемента:

Обратиться в CSS можно по названию элемента, например: div {}

По ID элемента:

Обратиться в CSS можно ID элемента, например: #id {}

По классу элемента:

Обратиться в CSS можно по названию элемента, например: .class {}

С помощью нескольких классов:

Обратиться в CSS можно по названию элемента, например: .class1, .class2 {}

По вложенности:

Обратиться можно по вложенности, например: header > h1 > a {}

Псевдоклассы:

Обратиться в CSS модно к псевдоклассам, например: .class:hover {}
